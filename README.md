# Recallify

Recallify is a simple AI-powered learning tool that transforms raw notes into structured content, flashcards, and quizzes.

## Features

* Convert messy notes into clean summaries
* Generate flashcards for active recal
* Create quiz questions for self-testing
* Simple and clean user interface

## Tech Stack

* React (Vite)
* Tailwind CSS
* Gemini API (for content generation)
* Vercel (deployment)

## Project Structure

```
src/
  components/
  screens/
  services/
```

## Setup

```bash
npm install
npm run dev
```

## Status

Initial setup complete. Core features in progress.

## Idea

The goal is to move from passive note-taking to active learning by automatically generating study materials from user input.
